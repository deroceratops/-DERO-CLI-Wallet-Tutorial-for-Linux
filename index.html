DERO CLI Wallet Tutorial for Linux

About

The DERO CLI Wallet is currently much more stable and have more features than the Engram GUI wallet.

The purpose of this guide is to show how the CLI wallet can be used on Linux.

Prerequisites

  • Have Linux installed on your computer.
  • Have basic familiarity with the terminal.

Installing the wallet CLI

  1. Open a terminal window. It's usually done by pressing CTRL + ALT + T.
  2. Go to your home directory using the terminal. Enter this command in the terminal to go there. cd
  3. Download the tar file containing the wallet executable. The URL is from the DERO Projects' official GitGub.
    4. wget https://github.com/deroproject/derohe/releases/latest/download/dero_linux_amd64.tar.gz
  4. Decompress the tar file. You will get a new directory called dero_linux_amd64. tar -xvzf dero_linux_amd64.tar.gz
  5. Enter the new directory. cd dero_linux_amd64
  6. Start the wallet CLI. ./dero-wallet-cli-linux-amd64 --remote You can optionally start it like this. It will also work if you are not in the wallet directory. /home/$USER/dero_linux_amd64/dero-wallet-cli-linux-amd64 --remote
  7. You will get output similar to this.

Create new wallet

  1. Go to your DERO wallet directory. cd /home/$USER/dero_linux_amd64
  2. Start the wallet CLI. ./dero-wallet-cli-linux-amd64 --remote
  3. In the menu type 2 for "Create New Wallet", then press ENTER.
  4. Type the filename you want (it's recommended to name it with .db as file extension, for example doe.db.) or leave blank to use the default filename, wallet.db. Then press ENTER.
  5. Choose a strong password and store it somewhere safe. Confirm your password.
  6. Press ENTER to get a seed with english words or choose another language.
  7. You will get 25 words. Write them down and store them in a safe place.
  8. Register your wallet by selecting number 4 in the menu, "Account registration to blockchain".
  9. Be patient and wait while your wallet is being registered.
  10. After registering you should see some new options in the menu.
  11. Show your address by selecting 1, "Display account Address" in the menu. The address is the long string beginning with "dero".
  12. You can verify that everything is working and you have the correct address by sending a small amount of DERO to your wallet. Be aware that it might take several minutes or more to withdraw DERO from an exchange.

Open a specific wallet if you have several wallets

If you have a wallet with the default name, wallet.db you will be prompted to enter it's password every time you start the CLI.

If you would like to open another wallet follow these steps.

  1. Go to your DERO wallet directory. cd /home/$USER/dero_linux_amd64
  2. List the wallet files in the directory. ls *.db
  3. Take note of the filename of the wallet you would like to open.
  4. Start the wallet CLI. ./dero-wallet-cli-linux-amd64 --remote
  5. If you have a wallet named wallet.db, you will be asked for it's password. Enter it. Otherwise go to step 7.
  6. Select 8, "Close Wallet".
  7. Select 1, "Open Existing Wallet".
  8. Enter the filename for the wallet you wish to open.
  9. Enter the wallet's password.

Recover wallet using recovery seed

  1. Go to your DERO wallet directory. cd /home/$USER/dero_linux_amd64
  2. Start the wallet CLI. ./dero-wallet-cli-linux-amd64 --remote
  3. Select 3, "Recover Wallet using recovery seed (25 words)".
  4. Enter the filename you want to use for the wallet. It's recommended to name it with .db as file extension, for example doe.db.
  5. Enter the password you would like to use to unlock the wallet and confirm.
  6. Enter your 25 word seed phrase.
  7. You should now get a message saying "Successfully recovered wallet from seed".
